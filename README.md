
#Hoppas

HTML (Jade), CSS (Stylus) Kit Starter.

## Поддержка браузеров
* Последние Chrome и Firefox
* IE10+
* Safari 7+

##Настройка gulp

0) Установить зависимости
 ```
npm i
 ```

1) Запустить gulp
 * ```gulp``` - Cобираем dev и запускаем локальный сервере
 * ```gulp dev``` - Сборка для разработки
 * ```gulp build``` - Релизная сборка
